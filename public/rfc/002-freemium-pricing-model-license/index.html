RFC-002
<DRAFT> Freemium pricing model for BIM software license

Motivation

The high price tag of the BIM softaware has been an issue for introducing the BIM process for many companies. This RFC aims to address that issue by requesting the software companies to introduce a freeminium pricing model like the modern SAASs to make the BIM software more accessible.

 

The high overhead cost and the limited number of the users and companies who can afford the software have also lead to a not-so-competetive software developer expereinces and services developed by them.

Current

Autodesk provides LT (lite) version of Revit license, however it doesn’t address a concern of the motivation for this RFC since it’s not very affordable and it doesn’t provide an access to the SDK.

Proposal

A rough outline of the proposal below.
